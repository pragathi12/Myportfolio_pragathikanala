# How to Add Your Resume PDF

## Quick Steps:

1. **Convert your resume to PDF** (if it's not already)
   - You can use any PDF converter or save as PDF from Word/Google Docs

2. **Place the PDF file** in the `public` folder:
   ```
   /Users/pragathikanala/Downloads/pragathi_portfolio/public/resume.pdf
   ```

3. **That's it!** The "Download Resume" button will automatically work.

## File Location:

Your resume PDF should be placed here:
```
/Users/pragathikanala/Downloads/pragathi_portfolio/public/resume.pdf
```

## How to Add It:

### Method 1: Using Finder
1. Open Finder
2. Navigate to: `/Users/pragathikanala/Downloads/pragathi_portfolio/public/`
3. Drag and drop your resume PDF into this folder
4. Make sure it's named exactly: `resume.pdf`

### Method 2: Using Terminal
```bash
# Copy your resume PDF to the public folder
cp "/path/to/your/resume.pdf" /Users/pragathikanala/Downloads/pragathi_portfolio/public/resume.pdf
```

## Important Notes:

- ✅ The file must be named: `resume.pdf` (lowercase)
- ✅ It must be in the `public/` folder
- ✅ After adding, refresh your browser to see the changes
- ✅ When someone clicks "Download Resume", it will download your PDF

## Testing:

After adding your resume:
1. Open your portfolio in the browser
2. Click the "Download Resume" button
3. Your PDF should download automatically!
