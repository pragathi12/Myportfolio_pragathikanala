# How to Add Your Photo

## Quick Steps:

1. **Find your photo file** on your computer
2. **Copy or move it** to the `public` folder in this project
3. **Rename it** to `photo.jpg` (or keep the original name if it's already jpg/jpeg/png)

## Supported Formats:
- `photo.jpg`
- `photo.jpeg`
- `photo.png`
- `photo.JPG`
- `photo.JPEG`
- `photo.PNG`

## Location:
Your photo should be placed here:
```
/Users/pragathikanala/Downloads/pragathi_portfolio/public/photo.jpg
```

## Tips:
- The photo will be displayed in a circular frame
- Recommended size: 500x500 pixels or larger (square images work best)
- The image will be automatically cropped to fit the circle
- If the photo doesn't load, a placeholder with "PK" will show instead

## After Adding:
1. Save the photo file in the `public` folder
2. Restart the dev server if it's running (stop with Ctrl+C, then run `npm run dev` again)
3. Refresh your browser to see your photo!
